# HubWorldMMO
OWS Hub World MMO Example Project

This project is currently using a [custom UE 5.7.4 version](https://github.com/jota2rz/UnrealEngine).

[Setup Instructions](https://www.hubworldmmo.com/)

## Fork
This repository is a fork of https://github.com/SabreDartStudios/HubWorldMMO.

It's not shown as a fork because GitHub doesn't allow new LFS files at public forks, and this repository contains everything needed to run the project, including content assets, which require binary file handling.

## MultiServer
The focus of this fork is to implement MultiServer functionality and allow seamless transfer between zones, you can read the plan at [MULTISERVER-PLAN.md](/MULTISERVER-PLAN.md).
